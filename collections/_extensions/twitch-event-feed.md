overview: |
  Twitch Event Feed is a SAMMI Core extension that creates a customizable Twitch event overlay for OBS using an HTML file as a browser source.

  The overlay displays the latest Twitch events in a clean animated feed, with support for avatars, custom colors, custom fonts, event labels, optional subline text, amounts, months, tiers, and automatic hiding after a configurable number of seconds.

  It is designed for streamers who want a clean, animated feed showing recent Twitch activity such as follows, bits, subscriptions, gifted subs, raids, channel points redemptions, automatic reward redemptions, watch streaks, custom power-ups and default power-ups.

  This extension is designed for streamers who want a lightweight, customizable Twitch activity feed controlled directly from SAMMI.

  #### Main Features
  - Animated overlay for recent Twitch events
  - Configurable event limit
  - Optional auto-hide timer
  - Cool animation effect when events disappear, are reset, or are removed by the event limit
  - Twitch avatar support with initial-letter fallback
  - Per-event customization for colors, labels, fonts, user names, sublines, messages, amounts, months, and tiers
  - Optional fields stay hidden when empty
  - Reset command with event-type filtering
  - SETUP button to create the overlay file and OBS scene/source

  #### +10 Included Commands
  - TEF: Add Follow
  - TEF: Add Bits
  - TEF: Add Sub
  - TEF: Add Gift Sub
  - TEF: Add Raid
  - TEF: Add Channel Points
  - TEF: Add Automatic Reward Redemption
  - TEF: Add Watch Streak
  - TEF: Add Custom Power-Up
  - TEF: Add Default Power-Up
  - TEF: Reset Feed

  #### Included Deck Buttons
  - Settings button to configure event limit, auto-hide timer, and enabled event types
  - Event example buttons for each supported Twitch event
  - Reset Feed button to clear all events or only a selected event type
  - SETUP button to create/update the overlay HTML and the OBS scene/source
